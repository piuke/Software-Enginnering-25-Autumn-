# 二次元网络商场系统 · 快速开始

一个基于 Python + SQLite 的命令行电商系统，支持用户、商品、消息、收藏与卖家管理，并完整支持多语言（简体中文 / English / 日本語）。

## 环境要求

- Python 3.8+（推荐 3.10 及以上）
- 操作系统：Windows / macOS / Linux（本项目在 Linux 下开发与验证）

可选（代码质量检查）：
- pip install -r requirements.txt（仅包含 pylint）

## 运行

```bash
cd exp3
python main.py
```

首次运行会在项目根目录自动创建 SQLite 数据库文件 `anime_mall.db`。

## 语言切换

- 在主菜单按 `L` 切换语言。
- 三种语言均已覆盖全部菜单、提示与错误信息。

## 核心功能

- 用户：注册、登录（可选择成为卖家）
- 商品：浏览（全部/分类/搜索/排序）、详情、收藏；卖家可发布/编辑/上下架
- 消息：
  - 联系人优先（Telegram 风格：按最新消息排序、未读徽标、卖家店铺标识）
  - 搜索用户后直接开始聊天
  - 商品详情页一键“联系卖家”
  - 打开会话即自动标记为已读（提供“刷新会话”）

## 目录结构（简）

```
exp3/
├── main.py           # 入口（菜单与交互）
├── models/           # 数据模型（User/Product/Message/...）
├── services/         # 业务逻辑（UserService/ProductService/...）
├── database/         # SQLite 初始化与管理
├── utils/            # 工具（校验、异常、哈希等）
└── config/           # 配置与多语言（translations.json）
```

## 常见问题（FAQ）

- 数据库在哪里？
  - 项目根目录 `anime_mall.db`，首次运行自动生成。
- 为什么不能给自己发消息？
  - 系统会拦截自发消息，防止误操作（并显示本地化提示）。
- 没看到“标记已读”？
  - 打开会话即自动标记为已读；如需手动刷新，使用“刷新会话”。
